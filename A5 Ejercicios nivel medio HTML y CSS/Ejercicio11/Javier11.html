Software Libre

Escritorio

7-zip

7-zip es un compresor/descompresor de archivos que maneja los formatos ZIP, RAR, CAB, GZIP, BZIP2 y TAR, así como un formato propio, 7z.

Audacity

Audacity es un editor de audio. Permite reproducir, grabar, editar, mezclar, aplicar efectos, etc.

GIMP

GIMP (GNU Image Manipulation Program) es un programa de retoque fotográfico y composición y edición de imágenes.

Inkscape

Inkscape es un editor de gráficos vectoriales que utiliza el formato SVG y que también importa formatos como JPG, EPS, PNG, y TIFF y exporta en PNG y varios formatos vectoriales.

LibreOffice

LibreOffice es una suite ofimática completa compuesta de editores de texto, hojas de cálculo, presentaciones, programa de dibujo, base de datos y editor de ecuaciones.

Internet

FileZilla

FileZilla es un cliente de FTP.

Firefox

Firefox es un navegador web altamente personalizable y ampliable mediante la gestión de extensiones y temas disponibles.

HTTrack

HTTrack permite descargar sitios web completos para su consulta offline.

Pidgin

Pidgin es un cliente de mensajería instantánea compatible con múltiples protocolos (Oscar y TOC/AIM, MSN Messenger, Yahoo, IRC, Jabber, Gadu-Gadu, Zephyr y otros).

Thunderbird

Thunderbird es un cliente de correo de los creadores de Firefox, que permite la gestión de múltiples cuentas de correo, la clasificación de mensajes mediante etiquetas y amplias posibilidades mediante extensiones.
